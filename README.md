# Задание «Игра с гоблинами»

## Легенда

Вы решили доделать игру с гоблинами, поэтому нужно реализовать оставшуюся логику, где гномы (или другие существа), выскакивают из "отверстий" и по ним нужно бить молотком:

![](https://github.com/danroman-github/ahj-dom/blob/main/src/img/GracefulMiniatureBustard-small.gif)

Copyright gfycat.com

### Описание

Нужно доделать игру с гоблинами, реализовав следующую логику:

1. Гоблин появляется в рандомной точке (набор точек фиксирован) ровно на 1 секунду
2. Если пользователь успел за это время кликнуть на этой точке (попробуйте сделать custom-курсор в виде молотка), то:
- пользователю засчитывается +1 балл
- гоблин пропадает из ячейки
3. Если пользователь пропустил 5 появлений гоблинов, то игра завершается.

Реализовать всё не в виде единого скрипта, а разбив приложение на классы, каждый из которых ответственен за опредённую логику.

Всё должно собираться через Webpack (включая картинки и стили) и выкладываться на Github Pages через CI.
